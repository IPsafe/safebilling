SafeBilling
===========

VoIP Billing and Routing platform

Based on FreeSWITCH


### Features
- Customers
- Customer Devices
- Dial Rules
- Providers
- Inbound Call Routes
- LCR (Least-cost routing)
- Routing based on area code
- Limit max calls by Gateway
- Call Rates
- Wholesale VOIP
- Business VOIP
- Residential VOIP
- CDR (Call Detail Record)
- Reports
- Blacklist
